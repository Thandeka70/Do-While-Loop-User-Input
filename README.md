# Do-While-Loop-User-Input
This is my Counter App.
The app ia a simple , educational android application designed to demonstrate how the "do-while" loop works.
A user is required to enter target numbers, the app generates a list counting from 1 up to the specified number, displaying the results in a cleen card based user interface.
i used EidtText , TextView and Button to create my user interface.
i used the do while loop logic to assit my app to work.
i used string and intergers as my data types for this interface.
